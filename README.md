## Installation

**Requirements**

C++ compiler (gcc 3.0 or higher)

**How to make**

```
aclocal && libtoolize --force && autoreconf
./configure
make
sudo make install
```

You can change default install path by using --prefix option of configure script.

Try --help option for finding out other options.
